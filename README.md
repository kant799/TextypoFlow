## 如何使用

在AiStudio中试用 [TextypoFlow](https://ai.studio/apps/drive/1Ct52k8D63yVIdI3tm9T9nh9qcA0Q7Bb0)

1. 右上角 Copy App 进行自定义的修改

<img width="300" alt="Image" src="https://github.com/user-attachments/assets/78b47522-3532-4d93-8767-45054a2af88f" /><br>

2. 添加自己的预设节点
   - 打开`code`Tab

   <img width="200" alt="Image" src="https://github.com/user-attachments/assets/0989c8f2-23ee-4851-b70d-b51f4933e67b" /><br>

   - 在`presets.tsx`文件内容添加新的预设节点
  
   <img height="160" alt="Image" src="https://github.com/user-attachments/assets/555bd43d-af79-4a6b-839e-f657e1cb8cda" /><br>

   - 在`\prompts`文件夹下添加`.txt`格式文件用于储存对应的新增系统提示词

   <img height="160" alt="Image" src="https://github.com/user-attachments/assets/296069df-5e37-4632-a869-1d07de917383" /><br>

   > 也可通过对话让AI添加
   > “帮我添加****的预设节点”
